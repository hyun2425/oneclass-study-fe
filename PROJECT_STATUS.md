# 프로젝트 현황 정리

## ✅ 완료된 기능

### 1. 온보딩 플로우
- [x] 온보딩 화면 (시작하기 버튼)
- [x] 사용자 타입 선택 (원생, 학부모, 일반)
- [x] 메인 화면으로 이동

### 2. 메인 화면 구조
- [x] 하단 탭바 3개 (홈, 그룹, 더보기)
- [x] 홈 화면 (대시보드 형태)
- [x] 그룹 목록 화면
- [x] 더보기 화면 (설정)

### 3. 그룹 기능
- [x] 그룹 목록 화면 (참여 중인 그룹 표시)
- [x] 그룹 만들기 화면
- [x] 그룹 상세 화면
- [x] 상위 3명 인증 순위 (포디엄 형태)
- [x] 전체 참여자 리스트
- [x] 인증 피드 (사진 + 텍스트)

## ⚠️ 추가/보완 필요 사항

### 1. 누락된 페이지
- [ ] **인증하기 페이지** (`/certify`)
  - GroupDetail에서 인증하기 버튼 클릭 시 이동하는 페이지가 없음
  - 사진 업로드 + 텍스트 입력 기능 (프로토타입 디자인만)

### 2. 사용자 타입별 차별화
- [ ] **원생/학부모/일반별 다른 홈 화면**
  - 현재는 "원생 대시보드"로 하드코딩되어 있음
  - SelectType에서 선택한 타입에 따라 다른 화면 표시 필요
  - 예: 학부모는 자녀 현황, 일반은 성인 스터디 중심

### 3. 네비게이션 개선
- [ ] **탭바 활성 상태 관리**
  - 현재 각 페이지에서 하드코딩된 `isActive` 상태
  - 현재 경로에 따라 동적으로 활성화되도록 개선 필요
  - 예: `/home`에서는 홈 탭 활성, `/group`에서는 그룹 탭 활성

### 4. 라우터 보완
- [ ] `/certify` 경로 추가 필요

### 5. UX 개선 사항
- [ ] 그룹 상세 페이지에서 뒤로가기 시 이전 그룹 목록으로 이동
- [ ] 인증 피드에서 날짜별 필터링 또는 정렬 기능 (프로토타입에서는 디자인만)
- [ ] 빈 상태 처리 (그룹이 없을 때, 인증이 없을 때 등)

## 📋 초기 요구사항 체크리스트

### ✅ 완료
1. ✅ 시작하기 클릭 시 원생/학부모/일반 선택
2. ✅ 선택 후 메인으로 이동
3. ✅ 메인 하단 탭바 3개 (홈, 그룹, 더보기)
4. ✅ 그룹 화면: 참여 그룹 목록 + 플러스 버튼
5. ✅ 그룹 만들기 화면
6. ✅ 그룹 상세: 상위 3명 순위, 경쟁자 리스트, 인증 피드
7. ✅ 더보기: 기본 설정 화면

### ⚠️ 부분 완료
1. ⚠️ 사용자 타입별 다른 홈 화면 (원생만 구현됨)
2. ⚠️ 인증하기 기능 (버튼은 있으나 페이지 없음)
